# 重生之我是姜云升

文字游戏，执行玩家角色姜云升，在2012年-2023年共计432个轮次中，体验一些人生中的小故事。

- [x] 写歌机制
- [x] 成就机制
- [ ] 物品机制

   常规升级方式：回家休息打游戏提升
